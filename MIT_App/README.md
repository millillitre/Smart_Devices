# MIT App

## Description

L'idée est de pouvoir utiliser une application directement sur notre smartphone pour obtenir les données de notre capteur de gaz.

### [TP test](/MIT_App/TP_test/)

Dans cette partie, nous essayons de faire fonctionner un TP sur une application MIT connecté à une carte arduino.

Dans la capture des [Blocks](/MIT_App/TP_test/Block.png), nous avons les blocks que nous avons utilisés, et dans la capture du [Screen](/MIT_App/TP_test/Screen.png) on peut voir la User Interface.

Le fichier [.aia](/MIT_App/TP_test/TP5_MOSH.aia) contient notre application.
